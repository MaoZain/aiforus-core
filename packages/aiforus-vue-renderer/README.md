# @aiforus/vue-renderer

Vue 3 renderer for the AIForUs OpenMAIC-compatible slide DSL.

This package is currently an unpublished `0.0.0` skeleton. Phase 3 implements
Slide rendering, playback actions, spotlight, laser, speech adapters, and
layout diagnostics. It deliberately does not expose or reproduce the React API
of `@openmaic/renderer`.

Quiz and Deep Interactive rendering are outside this package.
