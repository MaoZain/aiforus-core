# @aiforus/dsl

OpenMAIC-compatible slide data contracts maintained independently by AIForUs.

This package is currently an unpublished `0.0.0` skeleton. Phase 2 imports and
stabilizes the pinned MIT-licensed OpenMAIC DSL baseline, including types,
schemas, validators, normalizers, version helpers, migrations, fixtures, and
conformance tests.

It must remain free of runtime dependencies unless an explicit compatibility,
size, and license review approves a change.
